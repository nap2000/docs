Main Screen
-----------

The FieldTask main screen has three tabs:

Forms
+++++

Shows blank surveys that are available to complete. Select a survey to fill it in. When you finish, a new instance is
created and submitted to the server. The blank form remains available for reuse.

Tasks
+++++

Lists tasks assigned to the user, plus in-progress and completed instances.

For a description of task states, see :ref:`task-states`.

Submitted instances cannot be modified and will disappear after the next refresh.

An instance that is **finalized** (marked as complete but not submitted) cannot be edited, but you can review the data and
add notes. This review can be disabled using the **Mobile App Options** tab on the user management page.

Incomplete instances can be opened for editing. Long-press an incomplete instance or task to **reject** it. The instance is
deleted and you must provide a reason.

Map
+++

Shows task locations on the map. A short press shows the task name and opens a menu where you can use Google Maps to plan a
route. A long press opens the task for editing.
